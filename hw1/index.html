Task 1: Drawing Single-Color Triangles

In order to draw a continuous colored triangle to the screen space with discrete pixels, we need to sample if each pixel center is inside this triangle. In rasterize_triangle(), the basic idea is to use nested for loop to walk through each pixel, then if the center of a pixel is inside_triangle(x+0.5, y+0.5), fill_pixel() will be called. 


		//pseudocode
		for (x {0}; x < width; ++x)
			for (y {0}; y < height; ++y) {
				if (inside_triangle(tri, x+0.5, y+0.5)) {
					fill_pixel();}}
